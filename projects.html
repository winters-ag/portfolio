Bus Tracking Exercise

This is used for a project regarding tracking real time locations of busses around Boston, MA. It will update every 15 seconds with a new location of the busses. Previous locations are not stored, so once the map refreshes they are no longer available. This project will not run unless you follow the readme and create your own Google API Key.
